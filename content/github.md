Today I'm using [GitHub](https://github.com) to craft this website. My entire process takes place within the platform. The codebase for this site sits in a [repository on GitHub](https://github.com/micahwalter/micahwalter). (If you see a typo, please make a pull request!) I use GitHub's [Codespaces](/codespaces) to edit the code in my browser. Merging changes back to the main branch fires off a workflow that automatically deploys the new code to [AWS Amplify](/aws-amplify). It's all very nice and simple.

I think my favorite part is that everything is in the repository--tracked thoughts, branches, tags, and issues. It's thoughts on thoughts--meta-thoughts.
